# WayGo
Технічний опис проєкту "WayGo"
1. Загальна концепція та мета
WayGo — це інноваційний персоналізований вебзастосунок для пошуку локацій для відпочинку, що базується на емоційному контексті користувача.
Основна мета: Скоротити час користувача на прийняття рішення ("куди піти?") з 15-20 хвилин до декількох секунд.
Об’єкт розробки: Вебплатформа з адаптивним інтерфейсом, інтегрованою мапою та рекомендаційною системою.
Основна проблема: Перенасиченість ринку стандартними картографічними сервісами (Google Maps, TripAdvisor), які пропонують суху статистику (рейтинг, тип закладу), але не враховують миттєвий настрій людини.
Ключова інновація: Впровадження категорії "Вайб" (Vibe) як основного фільтра пошуку замість традиційних категорій.
2. Обсяг проекту та Функціональні можливості
2.1. Функціональні модулі
Персоналізований пошук: Фільтрація локацій за радіусом та "вайбом".
Інтерактивна мапа: Візуалізація місць у реальному часі з урахуванням геолокації.
Система "Вайбів": Класифікація місць за атмосферою (тиша, драйв, робота, романтика тощо).
Соціальна взаємодія: Можливість відмічати відвідування ("Я був тут"), залишати відгуки та лайки.
Історія та Обране: Формування персонального архіву вподобань для покращення майбутніх рекомендацій.
Онбордінг: Швидке знайомство з інтерфейсом та початкове налаштування профілю.
2.2. Функціональні модулі
Модуль ідентифікації та профілювання:
Авторизація через Google OAuth та Email.
Створення емоційного профілю користувача (аналіз історії вподобань).
Модуль "Vibe Engine" (Ядро вайбів):
Класифікація локацій за 7 напрямками: ToGo, Explore, Cozy, Active, Social, Nature, Special.
Динамічна зміна вайбу залежно від часу доби (наприклад, кав'ярня вранці — ToGo, ввечері — Cozy).
Геоінформаційний модуль:
Визначення координат через HTML5 Geolocation.
Візуалізація точок на Google Maps з кастомними маркерами.
Розрахунок радіуса доступності.
Інтелектуальна система рекомендацій:
Кнопка "Shuffle" для миттєвої зміни пропозицій.
Система взаємодії: "Я тут був", "Зберегти", "Лайк/Дизлайк".
2.3. Інтеграція Штучного Інтелекту (AI Core)
Це ключова технологічна перевага проєкту, що автоматизує збір даних:
NLP Аналіз: Використання LLM (GPT/Gemini) для сканування тисяч відгуків. ШІ виокремлює емоційні маркери ("затишно", "гучно", "інстаграмно") і автоматично присвоює вайб без участі модератора.
3. Технологічний стек
Frontend: React (SPA) + TypeScript.
Backend & Auth:  Node.js або Firebase
State Management: React Context / Custom Stores.
Maps: Інтеграція інтерактивних карт для візуалізації локацій.


Макет застосунку WayGo має дуже чистий, свіжий та «екологічний» вигляд. Основний акцент зроблено на легкості інтерфейсу (Light Mode) та природних відтінках зеленого, що асоціюється з навігацією, новими місцями та комфортом.
